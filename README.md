En este proyecto se continuara con la libreria comenzada anteriormente 
de operaciones basicas con números complejos. Para esto se 
realizaran diferentes funciones entre matrices, vectores y 
constantes (todas compuestas por números complejos). Las funciones
trabajadas son las siguientes:


1.Adición de vectores complejos: Los vectores deben tener la misma cantidad de filas para
poder operarlas, y se opera componente a componente.

2.Inverso (aditivo) de un vector complejo: En este se debe cambiar el símbolo de la real y la
parte imaginaria del número complejo (El vector se multiplica con -1).

3.Multiplicación de un escalar por un vector complejo: El esxalar se debe multiplicar con cada
uno de los elementos del vector.

4.Adición de matrices complejas: Deben tener el mismo tamaña (mxn) para sumarlas.

5.Inversa (aditiva) de una matriz compleja: Se debe hacer lo mismo que en el inverso aditivo
de los vectores.

6.Multiplicación de un escalar por una matriz compleja: Se multiplica una constante por una matriz.

7.Transpuesta de una matriz/vector: En esta se cambian las filas y las columnas, cambia el tamaño
de la matriz (mxn -> nxm).

8.Conjugada de una matriz/vector: En esta se debe cambiar el signo de la parte imaginaria de cada
número que compone la matriz o el vector.

9.Adjunta (daga) de una matriz/vector: Se debe sacar la adjunta de la matriz y la transpuesta
(El orden de las operaciones no afecta el resultado).

10.Producto de dos matrices (de tamaños compatibles): Se deben multiplica dos matrices con la misma cantidad
de filas y de columnas.

11.Función para calcular la "acción" de una matriz sobre un vector: Es la multiplicación de una matriz por un
vector (misma cantidad de columnas de la matriz y de filas del vector).

12.Producto interno de dos vectores: En este se debe sacar primero la daga del primer vector para posteriormente multiplicarlo 
por el segundo vector (dando así un número).

13.Norma de un vector: La norma de un vector se refiere al tamaño del mismo.En este se debe sacar primero la daga de el vector y posteriormente se multiplica este con el vector en su estado original, este número ya no es imaginario (solo queda
su parte real) y se le saca la raíz.

14.Distancia entre dos vectores: En este primero se restan dos vectores (del mismo tamaño) y a este se le saca la norma.

15.Revisar si una matriz es unitaria: Se considera una matriz unitaria cuando la multiplicación de esta con su
adjunta nos da un matriz unitaria (Está compuesta de ceros excepto su diagonal que se compone de unos).

16.Revisar si una matriz es Hermitiana: Se considera una matriz Hermitiana si su adjunta es igual a la matriz original.

17.Producto tensor de dos matrices/vectores: Es la multilicación de las dos matrices pero la matriz resultante 
queda con el tamaño (mxn)x(oxp) = filas(mxo) x columnas(nxp).

En esta liberia se trabajaran arreglos de la forma
[ [[a,b],[c,d]] , [[e,f],[g,h]] ] donde cada subocnjunto representa una fila, y los subconjuntos de los subconjuntos
representan las columnas.
